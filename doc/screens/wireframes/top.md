## URLパラメータ

| パラメータ名 |        パラメータ        |        説明        |               有効値               | パラメータ名なし時 |     対象外、範囲外の場合      |
| ------------ | ------------------------ | ------------------ | ---------------------------------- | ------------------ | ----------------------------- |
| page         | 数字                     | ページ指定         | 1 -                                | 1                  | 無効なURLであることを表示する |
| search       | tag:タグ名               | タグ名検索         | マンガに登録されているタグ         |                    | 検索結果0件を表示する         |
|              | series:シリーズ名        | シリーズ名検索     | マンガに登録されているシリーズ     |                    | 検索結果0件を表示する         |
|              | artist:アーティスト名    | アーティスト名検索 | マンガに登録されているアーティスト |                    | 検索結果0件を表示する         |
|              | character:キャラクター名 | キャラクター名検索 | マンガに登録されているキャラクター |                    | 検索結果0件を表示する         |
| order        | newer                    | 新しい順           | newer                              | newer              | 無効なURLであることを表示する |
|              | older                    | 古い順             | older                              |                    | 無効なURLであることを表示する |
|              | update                   | 更新順             | update                             |                    | 無効なURLであることを表示する |
|              | random                   | ランダム表示       | random                             |                    | 無効なURLであることを表示する |

- `&`で複数パラメータを同時設定可能。
  - ?page=1&order=older
  - AND検索 : ?search=tag:tag1^artist:artist1
  - OR検索  : ?search=tag:tag1|artist:artist1
