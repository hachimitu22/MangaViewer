# 用語辞書

|          用語          |    英語     |                     意味                     |
| ---------------------- | ----------- | -------------------------------------------- |
| メディア               | media       | 画像、動画のまとまり                         |
| メディア番号           | mediaId     | メディアの識別番号                           |
| メディアページ番号     | mediaPage   | メディアのページ番号                         |
| タグ                   | tag         | メディアを分類するタグ                       |
| カテゴリー             | category    | タグをカテゴライズするもの                   |
| アイテム               | item        | メディアの各画像、動画                       |
| アイテム番号           | itemId      | アイテムの識別番号                           |
| ビューアー             | viewer      | メディアを閲覧する画面                       |
| メディア一覧ページ番号 | summaryPage | メディア一覧のページ番号                     |
| お気に入り             | favorite    | ユーザーがお気に入りしたメディアのこと       |
| あとで見る             | queue       | ユーザーがあとで見るに追加したメディアのこと |
| ユーザー               | user        | アクターのユーザーを示す                     |
| 管理者                 | manager     | アクターの管理者を示す                       |
| 画面                   | screen      | ユーザーがブラウザで表示する画面             |
