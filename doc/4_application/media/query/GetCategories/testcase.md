# GetCategories テストケース

## 正常系

### 登録済みカテゴリー一覧を取得できる

- **前提**
  - 以下のカテゴリーが登録されている。
    - カテゴリー: "ジャンル"
    - カテゴリー: "年齢制限"
    - カテゴリー: "制作会社"
  - Repository は登録済みカテゴリー一覧を正常に返却する。

- **操作**
  - GetCategories を実行する。

- **結果**
  - 登録済みカテゴリー一覧が取得できる。
  - 取得結果にすべてのカテゴリーが含まれる。
  - 各カテゴリーは CategoryOutput（カテゴリー名）形式で返却される。

---

### 優先度順でカテゴリー一覧を取得できる

- **前提**
  - 以下のカテゴリーと優先度が登録されている。
    - 優先度1: "ジャンル"
    - 優先度2: "年齢制限"
    - 優先度3: "制作会社"
  - Repository は優先度順にソートされたカテゴリー一覧を返却する。

- **操作**
  - GetCategories を実行する。

- **結果**
  - カテゴリー一覧が優先度順で取得される。
  - 取得結果の順序が以下となる。
    1. "ジャンル"
    2. "年齢制限"
    3. "制作会社"
  - カテゴリーの順序が保持されている。

---

### カテゴリーが存在しない場合は空配列を返す

- **前提**
  - カテゴリーが1件も登録されていない。
  - Repository は空配列を返却する。

- **操作**
  - GetCategories を実行する。

- **結果**
  - エラーにはならない。
  - 空配列が返却される。
  - カテゴリー一覧の件数は 0 件である。

---

## 異常系

### Repository 取得に失敗した場合は失敗する

- **前提**
  - Repository がカテゴリー一覧取得時に例外またはエラーを返す。

- **操作**
  - GetCategories を実行する。

- **結果**
  - GetCategories は失敗する。
  - エラーが呼び出し元に返却される。
  - カテゴリー一覧は返却されない。
  - 永続化層のエラーがアプリケーションエラーとして扱われる。
