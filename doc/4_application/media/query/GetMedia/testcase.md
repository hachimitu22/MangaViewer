# GetMedia テストケース

## 正常系

- 指定したメディアIDの詳細を取得できる
  - 前提:
    - 指定IDのメディアが存在する
  - 操作:
    - GetMediaQuery を実行する
  - 期待結果:
    - MediaDetail DTO が返る

- タグ・カテゴリー・コンテンツ情報を含めて取得できる
  - 前提:
    - メディアにタグ・カテゴリー・コンテンツが登録されている
  - 操作:
    - GetMediaQuery を実行する
  - 期待結果:
    - すべての情報が DTO に含まれる

- 表示用DTO形式で取得できる
  - 期待結果:
    - ドメインエンティティや値オブジェクトを含まない

## 異常系

- 存在しないメディアIDの場合は取得できない
  - 前提:
    - 指定IDのメディアが存在しない
  - 操作:
    - GetMediaQuery を実行する
  - 期待結果:
    - 失敗結果が返る

- Repository 取得に失敗した場合は失敗する
  - 前提:
    - ReadRepository が例外を返す
  - 操作:
    - GetMediaQuery を実行する
  - 期待結果:
    - Query が失敗する
