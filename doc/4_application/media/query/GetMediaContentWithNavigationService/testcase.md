## 正常系

### 指定のコンテンツと前後のコンテンツを取得できる

- **前提**
  - 利用者が参照対象とするメディアが存在している。
  - 対象メディアには複数のコンテンツが登録されている。
  - 指定されたコンテンツはメディア内に存在している。
- **操作**
  - 利用者がメディアとコンテンツを指定して内容の参照を行う。
- **結果**
  - 指定されたコンテンツが取得できる。
  - 直前のコンテンツが取得できる。
  - 直後のコンテンツが取得できる。
  - メディア情報の取得処理が一度実行されている。
  - 指定したメディア識別子が取得処理に利用されている。
