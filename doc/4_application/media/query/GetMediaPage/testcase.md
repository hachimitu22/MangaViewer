# GetMediaPage テストケース

## 正常系

- 指定したページのコンテンツを取得できる
  - 前提:
    - メディアとページが存在する
  - 操作:
    - GetMediaPageQuery を実行する
  - 期待結果:
    - 指定ページのコンテンツDTOが返る

- 最初のページを取得できる
- 最後のページを取得できる

- ページング情報を取得できる
  - 期待結果:
    - 総ページ数・現在ページが正しい

## 境界・仕様確認

- ページ番号が範囲内であれば取得できる
- ページ番号が範囲外の場合は取得できない

## 異常系

- 存在しないメディアIDの場合は取得できない
- Repository 取得に失敗した場合は失敗する
