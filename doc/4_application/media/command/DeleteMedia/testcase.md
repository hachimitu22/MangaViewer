# DeleteMedia テストケース

## 正常系

### メディアを削除できる

- **前提**
  - 削除対象のメディアがリポジトリに存在する。
  - 管理者として認証済みである。

- **操作**
  - DeleteMedia に存在するメディアIDを指定して実行する。

- **結果**
  - 指定したメディアが削除される。
  - エラーは発生しない。

---

### 存在するメディアIDを指定した場合に削除が成功する

- **前提**
  - メディアID「media-001」に対応するメディアが存在する。
  - 管理者として認証済みである。

- **操作**
  - DeleteMediaInput に media-001 を指定して実行する。

- **結果**
  - media-001 のメディアがリポジトリから削除される。
  - 正常終了する。

---

## 異常系

### 指定されたメディアが存在しない場合は削除できない

- **前提**
  - 指定したメディアIDに対応するメディアがリポジトリに存在しない。
  - 管理者として認証済みである。

- **操作**
  - DeleteMedia に存在しないメディアIDを指定して実行する。

- **結果**
  - エラーが返却される。
  - メディアは削除されない。
  - 永続化処理は行われない。

---

### Repository で削除処理に失敗した場合は削除できない

- **前提**
  - 削除対象のメディアがリポジトリに存在する。
  - Repository の削除処理が失敗する状態である（例：DBエラー）。
  - 管理者として認証済みである。

- **操作**
  - DeleteMedia に存在するメディアIDを指定して実行する。

- **結果**
  - エラーが返却される。
  - メディアは削除されない。
  - 処理は異常終了する。
