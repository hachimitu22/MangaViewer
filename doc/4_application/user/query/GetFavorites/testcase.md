# GetFavorites テストケース

## お気に入り一覧を取得できる
- **前提**
  - ユーザーに複数のお気に入りメディアが登録されている
  - 認証済みである
- **操作**
  - GetFavoritesInput にユーザーIDを指定して実行する
- **結果**
  - お気に入りメディア一覧が取得できる
  - エラーは発生しない

---

## お気に入りが存在しない場合は空配列を返す
- **前提**
  - ユーザーのお気に入りが0件である
  - 認証済みである
- **操作**
  - GetFavoritesInput を指定して実行する
- **結果**
  - 空配列が返却される
  - エラーは発生しない

---

## 取得に失敗した場合はエラーとなる
- **前提**
  - 永続化レイヤーで取得処理が例外を返す
  - 認証済みである
- **操作**
  - GetFavorites を実行する
- **結果**
  - エラーが返却される
