## メディアをお気に入りに追加できる
- **前提**
  - 更新対象のユーザーが存在する
  - お気に入り追加対象のメディアが存在する
  - ユーザーのお気に入りに追加対象のメディアが含まれていない
- **操作**
  - お気に入りにメディアを追加する
- **結果**
  - お気に入り追加完了という結果が返却される
  - ユーザーリポジトリのユーザー取得処理にユーザーIDが指定されている。
  - メディアリポジトリのメディア取得処理にメディアIDが指定されている。
  - ユーザーリポジトリの保存処理にお気に入りにメディアが追加されたユーザーが指定されている。

---

## ユーザーが存在しない場合はユーザー未発見という結果となる
- **前提**
  - 更新対象のユーザーが存在しない
- **操作**
  - お気に入りにメディアを追加する
- **結果**
  - ユーザー未発見という結果が返却される。
  - ユーザーリポジトリのユーザー取得処理にユーザーIDが指定されている。
  - メディアリポジトリのメディア取得処理が実行されない。
  - ユーザーリポジトリの保存処理が実行されない。

---

## リポジトリの操作で例外が発生したらエラーとする
- **前提**
  - 更新対象のユーザーが存在する
  - お気に入り追加対象のメディアが存在する
  - ユーザーのお気に入りに追加対象のメディアが含まれていない
  - 永続化処理が失敗する状況（ストレージ・DB障害）
- **操作**
  - お気に入りにメディアを追加する
- **結果**
  - エラーが返却される
  - ユーザーリポジトリのユーザー取得処理にユーザーIDが指定されている。
  - メディアリポジトリのメディア取得処理にメディアIDが指定されている。
  - ユーザーリポジトリの保存処理にお気に入りにメディアが追加されたユーザーが指定されている。
