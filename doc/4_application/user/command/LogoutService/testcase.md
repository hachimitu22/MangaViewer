# LogoutService テストケース

## テストケース一覧

- [ログアウトに成功した場合はログアウト成功結果が返却される](#ログアウトに成功した場合はログアウト成功結果が返却される)
- [セッションが未設定の場合はログアウト失敗結果が返却される](#セッションが未設定の場合はログアウト失敗結果が返却される)
- [終了処理が失敗した場合は例外として扱う](#終了処理が失敗した場合は例外として扱う)

---

## ログアウトに成功した場合はログアウト成功結果が返却される
- **前提**
  - セッションに有効なセッショントークンが設定されている
  - `sessionTerminator` が `true` を返却する
- **操作**
  - ログアウトを実行する
- **結果**
  - ログアウト成功結果が返却される
  - 結果の `code` は `0` である
  - `sessionTerminator.execute` にセッションが指定されている

---

## セッションが未設定の場合はログアウト失敗結果が返却される
- **前提**
  - セッションに `session_token` が設定されていない、またはセッションストア上に削除対象が存在しない
  - `sessionTerminator` が `false` を返却する
- **操作**
  - ログアウトを実行する
- **結果**
  - ログアウト失敗結果が返却される
  - 結果の `code` は `1` である
  - `sessionTerminator.execute` にセッションが指定されている

---

## 終了処理が失敗した場合は例外として扱う
- **前提**
  - セッションに有効なセッショントークンが設定されている
  - `sessionTerminator` のセッション破棄処理または依存ストア操作で障害が発生する
- **操作**
  - ログアウトを実行する
- **結果**
  - 例外となる
  - `sessionTerminator.execute` にセッションが指定されたうえで失敗する
  - ログアウト成功結果およびログアウト失敗結果は返却されない
