## メディアをあとで見るに追加できる
- **前提**
  - 更新対象のユーザーが存在する
  - あとで見る追加対象のメディアが存在する
  - ユーザーのあとで見るに追加対象のメディアが含まれていない
- **操作**
  - あとで見るにメディアを追加する
- **結果**
  - あとで見る追加完了という結果が返却される
  - ユーザーリポジトリのユーザー取得処理にユーザーIDが指定されている。
  - メディアリポジトリのメディア取得処理にメディアIDが指定されている。
  - ユーザーリポジトリの保存処理にあとで見るにメディアが追加されたユーザーが指定されている。

## ユーザーが存在しない場合はユーザー未発見という結果となる
- **前提**
  - 更新対象のユーザーが存在しない
- **操作**
  - あとで見るにメディアを追加する
- **結果**
  - ユーザー未発見という結果が返却される。
  - ユーザーリポジトリのユーザー取得処理にユーザーIDが指定されている。
  - メディアリポジトリのメディア取得処理が実行されない。
  - ユーザーリポジトリの保存処理が実行されない。

## リポジトリの操作で例外が発生したらエラーとする
- **前提**
  - 更新対象のユーザーが存在する
  - あとで見る追加対象のメディアが存在する
  - ユーザーのあとで見るに追加対象のメディアが含まれていない
  - 永続化処理が失敗する状況（ストレージ・DB障害）
- **操作**
  - あとで見るにメディアを追加する
- **結果**
  - エラーが返却される
  - ユーザーリポジトリのユーザー取得処理にユーザーIDが指定されている。
  - メディアリポジトリのメディア取得処理にメディアIDが指定されている。
  - ユーザーリポジトリの保存処理にあとで見るにメディアが追加されたユーザーが指定されている。
