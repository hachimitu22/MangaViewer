# RemoveQueue テストケース

## あとで見るからメディアを削除できる
- **前提**
  - メディアがあとで見るに含まれている
- **操作**
  - RemoveQueueInput を指定して実行
- **結果**
  - メディアがあとで見るから削除される
  - エラーは発生しない

---
