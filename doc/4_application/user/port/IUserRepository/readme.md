# IUserRepository

## 概要
ユーザー永続化の抽象契約。

---

## テスト方針
- interface 自体はテスト対象にせず、`SequelizeUserRepository` などの具象実装で favorite / queue の保存・更新・未登録ユーザー・異常系を検証する。
- テストケース文書も具象実装または契約テスト粒度に合わせ、interface 文書には責務と共通契約だけを残す。
- 実装差し替えが発生した場合は、「未登録 user_id で空の `User` を返す」などを契約テストとして再利用する。

---

## 機能
### 保存
ユーザーを永続化する。

#### 入力
- ユーザー

#### 出力
- なし

---

### ユーザーIDからユーザーを取得
リポジトリからユーザーIDでユーザーを取得する。
ユーザーIDと一致するユーザーが存在しない場合は何も返さない。

#### 入力
- ユーザーID

#### 出力
- ユーザー
