# メディア一覧画面

## 画面説明
- メディア一覧を表示する。
- 検索していた場合は検索条件も表示する。
- サムネイル、タイトルを押下した場合はメディア詳細画面へ遷移する。
- タグを押下した場合は、タグを検索条件としてメディア一覧に遷移する。
- ページネーションは押下されたページ番号のメディア一覧に遷移する。

## URL
/summary

## クエリパラメータ
title=羅生門&tags=作者:太宰治&tags=種類:小説&sort=date_asc

  | 変数名 |            値             |                説明                |
  | ------ | ------------------------- | ---------------------------------- |
  | title  | {文字列}                  | タイトルを部分一致で検索する文字列 |
  | tags   | {カテゴリー名}:{ラベル名} | 検索するタグ、複数指定可能         |
  | sort   | {数値}                    | ソート条件を数値で指定する         |

### sort

|     値     |      説明      |
| ---------- | -------------- |
| date_asc   | 登録日の昇順   |
| date_desc  | 登録日の降順   |
| title_asc  | タイトルの昇順 |
| title_desc | タイトルの降順 |
| random     | ランダム       |

指定なし、上記以外の値は`date_asc`を指定していることと同じとする

## APIエラー時の表示方針
- 本画面固有の更新系APIは使用しない。`POST /api/logout`（ナビゲーターのログアウト）がエラー系レスポンスを返した場合は、画面上にエラーメッセージを表示し、ユーザーに再度実行を促す。
