# イベント一覧

| 要素No |             要素名             | トリガー | 動作 |
| ------ | ------------------------------ | -------- | ---- |
| 5      | カテゴリードロップダウンリスト | 選択時   | タグドロップダウンリストに選択したカテゴリーのタグ一覧を設定する。 |
| 9      | 追加ボタン                     | 押下時   | カテゴリー、タグのドロップダウンリストで選択しているものをタグ欄として追加する。 |
| 13     | 検索ボタン                     | 押下時   | タイトル、タグ欄、並び順を検索条件としてメディア一覧画面に遷移する。遷移時のページは1ページ目とし、並び順は並び順ドロップダウンリストの5種の選択肢から選ばれた値をそのまま使用する。 |
