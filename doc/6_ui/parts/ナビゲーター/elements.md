# 要素一覧

1. ナビゲーターエリア
    - 概要: ナビゲーターの表示エリア。
2. メディア一覧リンク
    - 概要: メディア一覧画面へのリンク
    - ラベル: "メディア一覧"
3. お気に入りリンク
    - 概要: お気に入り画面へのリンク
    - ラベル: "お気に入り"
4. あとで見るリンク
    - 概要: あとで見る画面へのリンク
    - ラベル: "あとで見る"
5. メディア登録リンク
    - 概要: メディア登録画面へのリンク
    - ラベル: "メディア登録"
    - 表示条件: 管理者がログインしている場合のみ表示する。
6. ログアウトボタン
    - 概要: ログアウトを行う
    - ラベル: "ログアウト"
