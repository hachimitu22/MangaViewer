# イベント一覧

| 要素No |                要素名                |      トリガー      |                                       動作                                       |
| ------ | ------------------------------------ | ------------------ | -------------------------------------------------------------------------------- |
| 7      | カテゴリー入力欄                     | 選択時             | 選択されたカテゴリーのタグ一覧をタグ入力欄に設定する。                           |
| 8      | タグ追加ボタン                       | 押下時             | カテゴリー入力欄、タグ入力欄に入力したものをタグ欄に追加する。                   |
| 11     | ファイル選択エリア                   | 押下時             | ファイル選択ダイアログを表示し選択されたファイルをメディアファイル欄に追加する。 |
| 11     | ファイル選択エリア                   | ファイルドロップ時 | ドロップされたファイルをメディアファイル欄に追加する。                           |
| 13     | メディアファイル削除ボタン           | 押下時             | 該当のメディアファイルを削除する。                                               |
| 14     | メディアファイル番号入れ替え上ボタン | 押下時             | 該当のメディアファイルと1つ上のメディアファイルの順番を入れ替える。              |
| 16     | メディアファイル番号入れ替え下ボタン | 押下時             | 該当のメディアファイルと1つ下のメディアファイルの順番を入れ替える。              |
| 20     | メディア更新ボタン                   | 押下時             | タイトル、タグ、メディアファイルでメディアを更新する。                           |
