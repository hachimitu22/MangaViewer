# ユビキタス言語

|          用語          |      英語       |                        意味                        |
| ---------------------- | --------------- | -------------------------------------------------- |
| メディア               | media           | コンテンツを順序付きで保持し、タグで分類される     |
| コンテンツ             | content         | メディアに含まれる１件の表示単位                   |
| タグ                   | tag             | メディアを分類するためのラベル                     |
| カテゴリー             | category        | タグを分類する区分                                 |
| ラベル                 | label           | カテゴリー内でタグを識別する名称                   |
| ユーザー               | user            | メディアを利用する主体                             |
| お気に入り             | favorite        | ユーザーが登録したメディアの集合                   |
| あとで見る             | queue           | ユーザーが後で閲覧するために保持するメディアの集合 |
|                        |                 |                                                    |
| メディアID             | mediaId         | メディアを一意に識別する識別子                     |
| ユーザーID             | userId          | ユーザーを一意に識別する識別子                     |
| タグID                 | tagId           | タグを一意に識別する識別子                         |
|                        |                 |                                                    |
| お気に入りに追加する   | add favorite    | メディアをお気に入りに登録する                     |
| お気に入りから削除する | remove favorite | メディアをお気に入りから外す                       |
| あとで見るに追加する   | add queue       | メディアをあとで見るに追加する                     |
| あとで見るから削除する | remove queue    | メディアをあとで見るから外す                       |
| タグを付与する         | attach tag      | メディアにタグを関連付ける                         |
| タグを外す             | detach tag      | メディアからタグを外す                             |
| コンテンツを編集する   | edit contents   | メディア内のコンテンツ構成を変更する               |
