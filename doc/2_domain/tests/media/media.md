# メディア集約 ― メディア（Aggregate Root）

## メディアの生成に成功する
- 前提
  - 有効なメディアIDが与えられている
  - 制御文字を含まないタイトルが与えられている
  - コンテンツが1件以上存在する
- 操作
  - メディアを生成する
- 結果
  - メディアが生成される
  - 指定したID・タイトル・コンテンツを保持する

## コンテンツが0件のためメディアの生成に失敗する
- 前提
  - 有効なメディアIDが与えられている
  - 有効なメディアタイトルが与えられている
  - コンテンツが0件である
- 操作
  - メディアを生成する
- 結果
  - メディアは生成されない
  - 「コンテンツは1件以上必要」という不変条件に違反する

## コンテンツの順序はメディアが管理する
- 前提
  - 複数のコンテンツを保持するメディアが存在する
- 操作
  - メディアに対してコンテンツ構成の編集を行う
- 結果
  - コンテンツの順序が変更される

## タグを関連付けることができる
- 前提
  - タグが関連付けられていないメディアが存在する
- 操作
  - メディアにタグを1件関連付ける
- 結果
  - メディアはタグを1件保持する
  - カテゴリー優先度はタグを関連付けた順となる

## 同一ではないタグを複数関連付けることができる
- 前提
  - すでに1件のタグが関連付けられているメディアが存在する
- 操作
  - 同一ではないタグを追加で関連付ける
- 結果
  - メディアはタグを2件保持する
  - カテゴリー優先度はタグを関連付けた順となる

## 同一タグを重複して関連付けることはできない
- 前提
  - すでに1件のタグが関連付けられているメディアが存在する
- 操作
  - 同一カテゴリー・同一ラベルのタグを再度関連付ける
- 結果
  - タグは追加されない
  - 「同一タグを重複して関連付けられない」という不変条件に違反する
  - カテゴリー優先度はタグを関連付けた順となる

## カテゴリー優先度を明示的に設定できる
- 前提
  - タグが関連付けられていないメディアが存在する
- 操作
  - カテゴリー優先度を設定する
- 結果
  - カテゴリー優先度順は設定した通りとなる

## タグの関連付け後にカテゴリー優先度を設定できる
- 前提
  - タグが関連付けられているメディアが存在する
- 操作
  - カテゴリー優先度を設定する
- 結果
  - タグは指定されたカテゴリー優先度順で扱われる

## カテゴリー優先度が未指定のカテゴリーはタグの関連付け順を優先度とする
- 前提
  - 複数カテゴリーのタグが関連付けられているメディアが存在する
- 操作
  - 1つだけカテゴリー優先度を設定する
- 結果
  - タグは指定されたカテゴリー優先度順で扱われる
  - カテゴリー優先度が指定されていないカテゴリーはタグ関連付け順を優先度として扱われる
