# ラベル（Value Object）

## ラベルの生成に成功する
- 前提
  - 制御文字を含まない文字列が与えられている
- 操作
  - ラベルを生成する
- 結果
  - ラベルが生成される

## 制御文字を含むため生成に失敗する
- 前提
  - 制御文字を含む文字列が与えられている
- 操作
  - ラベルを生成する
- 結果
  - ラベルは生成されない

## 同一ラベルは同一と判定される
- 前提
  - 同じ値を持つ2つのラベルが存在する
- 操作
  - ラベル同士を比較する
- 結果
  - 同一と判定される

## 異なる値のラベルは同一と判定されない
- 前提
  - 異なる値を持つ2つのラベルが存在する
- 操作
  - ラベル同士を比較する
- 結果
  - 同一とは判定されない
