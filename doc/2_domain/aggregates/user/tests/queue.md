# あとで見る（Value Object）

## あとで見るの生成に成功する
- 前提
  - 有効なメディアIDを与えられている
- 操作
  - あとで見るを生成する
- 結果
  - あとで見るが生成される

## あとで見るの生成に失敗する
- 前提
  - 無効なメディアIDを与えられている
- 操作
  - あとで見るを生成する
- 結果
  - あとで見るは生成されない

## 同一のメディアIDを持つあとで見るは同一と判定される
- 前提
  - 同じメディアIDを持つ2つのあとで見るが存在する
- 操作
  - あとで見る同士を比較する
- 結果
  - 同一と判定される

## 異なるメディアIDを持つあとで見るは同一と判定されない
- 前提
  - 異なるメディアIDを持つ2つのあとで見るが存在する
- 操作
  - あとで見る同士を比較する
- 結果
  - 同一とは判定されない
