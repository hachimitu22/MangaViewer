# ユーザーID（Value Object）

## ユーザーIDの生成に成功する
- 前提
  - 有効なIDが与えられている
- 操作
  - ユーザーIDを生成する
- 結果
  - ユーザーIDが生成される
  - 指定した文字列を保持する

## ユーザーIDの生成に失敗する
- 前提
  - 無効なIDが与えられている
- 操作
  - ユーザーIDを生成する
- 結果
  - ユーザーIDは生成されない

## 同じ値のユーザーIDは同一と判定される
- 前提
  - 同じ値を持つ2つのユーザーIDが存在する
- 操作
  - ユーザーID同士を比較する
- 結果
  - 2つのユーザーIDは同一と判定される

## 異なる値のユーザーIDは同一と判定されない
- 前提
  - 異なる値を持つ2つのユーザーIDが存在する
- 操作
  - ユーザーID同士を比較する
- 結果
  - 同一とは判定されない
