# お気に入り（Value Object）

## お気に入りの生成に成功する
- 前提
  - 有効なメディアIDを与えられている
- 操作
  - お気に入りを生成する
- 結果
  - お気に入りが生成される

## お気に入りの生成に失敗する
- 前提
  - 無効なメディアIDを与えられている
- 操作
  - お気に入りを生成する
- 結果
  - お気に入りは生成されない

## 同一のメディアIDを持つお気に入りは同一と判定される
- 前提
  - 同じメディアIDを持つ2つのお気に入りが存在する
- 操作
  - お気に入り同士を比較する
- 結果
  - 同一と判定される

## 異なるメディアIDを持つお気に入りは同一と判定されない
- 前提
  - 異なるメディアIDを持つ2つのお気に入りが存在する
- 操作
  - お気に入り同士を比較する
- 結果
  - 同一とは判定されない
