# カテゴリー（Value Object）

## カテゴリーの生成に成功する
- 前提
  - 制御文字を含まない文字列が与えられている
- 操作
  - カテゴリーを生成する
- 結果
  - カテゴリーが生成される

## 制御文字を含むため生成に失敗する
- 前提
  - 制御文字を含む文字列が与えられている
- 操作
  - カテゴリーを生成する
- 結果
  - カテゴリーは生成されない

## trim後に空文字となるため生成に失敗する
- 前提
  - 空文字、または空白文字のみの文字列が与えられている
- 操作
  - カテゴリーを生成する
- 結果
  - カテゴリーは生成されない

## 同一カテゴリーは同一と判定される
- 前提
  - 同じ値を持つ2つのカテゴリーが存在する
- 操作
  - カテゴリー同士を比較する
- 結果
  - 同一と判定される

## 異なる値のカテゴリーは同一と判定されない
- 前提
  - 異なる値を持つ2つのカテゴリーが存在する
- 操作
  - カテゴリー同士を比較する
- 結果
  - 同一とは判定されない
