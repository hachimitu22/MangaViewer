# メディアタイトル（Value Object）

## メディアタイトルの生成に成功する
- 前提
  - 制御文字を含まない文字列が与えられている
- 操作
  - メディアタイトルを生成する
- 結果
  - メディアタイトルが生成される
  - 指定した文字列を保持する

## 制御文字を含むため生成に失敗する
- 前提
  - 制御文字を含む文字列が与えられている
- 操作
  - メディアタイトルを生成する
- 結果
  - メディアタイトルは生成されない

## 同一メディアタイトルは同一と判定される
- 前提
  - 同じ値を持つ2つのメディアタイトルが存在する
- 操作
  - メディアタイトル同士を比較する
- 結果
  - 同一と判定される

## 異なる値のメディアタイトルは同一と判定されない
- 前提
  - 異なる値を持つ2つのメディアタイトルが存在する
- 操作
  - メディアタイトル同士を比較する
- 結果
  - 同一とは判定されない
