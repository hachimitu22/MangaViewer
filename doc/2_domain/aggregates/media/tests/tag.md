# タグ（Value Object）

## タグの生成に成功する
- 前提
  - 有効なカテゴリーが与えられている
  - 有効なラベルが与えられている
- 操作
  - タグを生成する
- 結果
  - タグが生成される
  - カテゴリーとラベルを保持する

## タグの生成に失敗する（カテゴリーが無効）
- 前提
  - 無効なカテゴリーが与えられている
  - 有効なラベルが与えられている
- 操作
  - タグを生成する
- 結果
  - タグは生成されない
  - カテゴリーが有効である必要があるという制約に違反する

## タグの生成に失敗する（ラベルが無効）
- 前提
  - 有効なカテゴリーが与えられている
  - 無効なラベルが与えられている
- 操作
  - タグを生成する
- 結果
  - タグは生成されない
  - ラベルが有効である必要があるという制約に違反する

## タグの生成に失敗する（カテゴリー・ラベルのいずれも無効）
- 前提
  - 無効なカテゴリーが与えられている
  - 無効なラベルが与えられている
- 操作
  - タグを生成する
- 結果
  - タグは生成されない

## 同一カテゴリー・同一ラベルのタグは同一と判定される
- 前提
  - 同じカテゴリー・ラベルの組み合わせを持つ2つのタグが存在する
- 操作
  - タグ同士を比較する
- 結果
  - 同一と判定される

## カテゴリーが異なるタグは同一と判定されない
- 前提
  - カテゴリーが異なるタグが存在する
- 操作
  - タグ同士を比較する
- 結果
  - 同一とは判定されない

## ラベルが異なるタグは同一と判定されない
- 前提
  - ラベルが異なるタグが存在する
- 操作
  - タグ同士を比較する
- 結果
  - 同一とは判定されない

## カテゴリーとラベルどちらも異なるタグは同一と判定されない
- 前提
  - カテゴリーとラベルがどちらも異なるタグが存在する
- 操作
  - タグ同士を比較する
- 結果
  - 同一とは判定されない
