# MediaViewer 仕様書

## プロジェクト概要
MediaViewerは、漫画・動画などの複数種類のメディアを閲覧可能なWebアプリケーションです。

### 技術スタック
- Node.js + Express
- EJSテンプレート
- Sequelize ORM
- SQLite3（開発）
- PostgreSQL（本番）
- デプロイ先：Raspberry Pi（予定）

詳細は `requirements/` および `design/` を参照してください。
