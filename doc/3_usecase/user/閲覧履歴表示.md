# 閲覧履歴表示

## 概要
ユーザーがメディア詳細を開いたメディアの履歴を表示する

## アクター
- ユーザー

## 事前条件
- ユーザーがログインしている
- メディア詳細を1件以上開いたことがある

## 事後条件
- 閲覧履歴が画面に表示される

## トリガー
- ユーザーが閲覧履歴を開く

## 基本フロー
1. ユーザーは閲覧履歴を表示する
2. システムはユーザーの閲覧履歴を取得する
3. システムは「閲覧日時の新しい順」で閲覧履歴を表示する

## 代替フロー
- ユーザーの閲覧履歴が0件の場合、「閲覧履歴が0件」のメッセージを表示する

## 関連ユースケース
- メディア詳細表示（事後）
