# お気に入り追加

## 概要
ユーザーがメディアをお気に入り一覧に追加する

## アクター
- ユーザー

## 事前条件
- ユーザーがログインしている

## 事後条件
- 該当メディアがお気に入り一覧に追加されている

## トリガー
- ユーザーがお気に入りへの追加を指示する

## 基本フロー
1. ユーザーはメディア一覧またはメディア詳細からお気に入りを指示する
2. システムはユーザーのお気に入り一覧に当該メディアを追加する

## 代替フロー
- お気に入りの追加に失敗した場合、「エラーが発生しました」のメッセージを表示する

## 関連ユースケース
- メディア一覧表示（事前・事後）
- メディア詳細表示（事前・事後）
- お気に入り一覧表示（事前・事後）
