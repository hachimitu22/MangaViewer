# お気に入り一覧表示

## 概要
ユーザーがお気に入りに追加したメディアの一覧を画面に表示する。
検索条件が指定された場合は検索した結果を表示する。

## アクター
- ユーザー

## 事前条件
- ユーザーがログインしている
- お気に入りを1件以上登録している

## 事後条件
- お気に入り一覧が画面に表示される
- 一覧からメディアを選択すると詳細画面に遷移する

## トリガー
- ユーザーがお気に入り一覧を開く

## 基本フロー
1. ユーザーはお気に入り一覧画面を表示する
2. システムはユーザーのお気に入り一覧を取得する
3. システムは「お気に入り追加日時の新しい順」でお気に入り一覧を表示する
4. ユーザーは任意のメディアを選択して詳細画面に遷移する

## 代替フロー
- ユーザーがお気に入りが0件の場合、「お気に入りが0件」のメッセージを表示する

## 関連ユースケース
- お気に入り削除（事後）
