# キュー一覧表示

## 概要
ユーザーがキューに追加したメディアの一覧を画面に表示する。

## アクター
- ユーザー

## 事前条件
- ユーザーがログインしている
- キューを1件以上登録している

## 事後条件
- キュー一覧が画面に表示される
- 一覧からメディアを選択すると詳細画面に遷移する

## トリガー
- ユーザーがキュー一覧を開く

## 基本フロー
1. ユーザーはキュー一覧画面を表示する
2. システムはユーザーのキュー一覧を取得する
3. システムは「キュー追加日時の新しい順」でキュー一覧を表示する

## 代替フロー
- ユーザーのキューが0件の場合、「キューが0件」のメッセージを表示する

## 関連ユースケース
- キュー削除（事後）
