## 正常系

### メディア登録が成功する

- **前提**
  - 有効なセッションIDがCookieに設定されている。
  - `title` が指定されている。
  - `contents` に `file` と `position` を持つ要素が1件以上指定されている。
  - `tags` に `category` と `label` を持つ要素が指定されている。
- **操作**
  - `multipart/form-data` で `POST /api/media` を実行する。
- **結果**
  - HTTPステータス `200` を返す。
  - レスポンスボディとして `code: 0` および `mediaId` を返す。

---

### タグが未指定でもメディア登録が成功する

- **前提**
  - 有効なセッションIDがCookieに設定されている。
  - `title` が指定されている。
  - `contents` に `file` と `position` を持つ要素が1件以上指定されている。
  - `tags` は未指定、または空配列である。
- **操作**
  - `multipart/form-data` で `POST /api/media` を実行する。
- **結果**
  - HTTPステータス `200` を返す。
  - レスポンスボディとして `code: 0` および `mediaId` を返す。

---

### タグが重複していてもメディア登録が成功する

- **前提**
  - 有効なセッションIDがCookieに設定されている。
  - 有効な `title` と `contents` が指定されている。
  - `tags` に同一の `category` + `label` の組み合わせが重複して含まれる。
- **操作**
  - `multipart/form-data` で `POST /api/media` を実行する。
- **結果**
  - HTTPステータス `200` を返す。
  - レスポンスボディとして `code: 0` および `mediaId` を返す。

---

## 異常系

### セッションIDが未指定の場合は401を返す

- **前提**
  - セッションIDをCookieに設定しない。
- **操作**
  - `multipart/form-data` で `POST /api/media` を実行する。
- **結果**
  - HTTPステータス `401` を返す。

---

### セッションが無効の場合は401を返す

- **前提**
  - 無効または期限切れのセッションIDがCookieに設定されている。
- **操作**
  - `multipart/form-data` で `POST /api/media` を実行する。
- **結果**
  - HTTPステータス `401` を返す。

---

### コンテンツ一覧が空の場合は登録失敗を返す

- **前提**
  - 有効なセッションIDがCookieに設定されている。
  - `title` は有効値が指定されている。
  - `contents` が空配列である。
- **操作**
  - `multipart/form-data` で `POST /api/media` を実行する。
- **結果**
  - HTTPステータス `200` を返す。
  - レスポンスボディとして `code: 1` を返す。

---

### コンテンツ保存に失敗した場合は登録失敗を返す

- **前提**
  - 有効なセッションIDがCookieに設定されている。
  - 入力値はすべて有効である。
  - ストレージへのコンテンツ保存で失敗が発生する。
- **操作**
  - `multipart/form-data` で `POST /api/media` を実行する。
- **結果**
  - HTTPステータス `200` を返す。
  - レスポンスボディとして `code: 1` を返す。

---

### メディア永続化に失敗した場合は登録失敗を返す

- **前提**
  - 有効なセッションIDがCookieに設定されている。
  - 入力値はすべて有効である。
  - データベースへのメディア保存で失敗が発生する。
- **操作**
  - `multipart/form-data` で `POST /api/media` を実行する。
- **結果**
  - HTTPステータス `200` を返す。
  - レスポンスボディとして `code: 1` を返す。
