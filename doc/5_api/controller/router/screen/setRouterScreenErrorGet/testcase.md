### GET /screen/error に描画ハンドラーを登録する
- **前提**
  - `router.get` をモック化する。
- **操作**
  - `setRouterScreenErrorGet` を実行する。
- **結果**
  - `router.get` が1回呼ばれる。
  - 第1引数が `/screen/error` である。
  - 第2引数に1つのハンドラーが設定される。

### 登録済みハンドラーを実行するとエラー画面を描画する
- **前提**
  - レスポンスオブジェクトで `status` / `render` をモック化する。
- **操作**
  - 登録済みハンドラーを実行する。
- **結果**
  - `screen/error` が案内メッセージと遷移リンクを含む表示データで描画される。
