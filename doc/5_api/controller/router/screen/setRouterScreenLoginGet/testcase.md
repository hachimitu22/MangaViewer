### GET /screen/login に描画ハンドラーを登録する
- **前提**
  - `router.get` をモック化する。
- **操作**
  - `setRouterScreenLoginGet` を実行する。
- **結果**
  - `router.get` が1回呼ばれる。
  - 第1引数が `/screen/login` である。
  - 第2引数に1つのハンドラーが設定される。

---

### 登録済みハンドラーを実行するとログイン画面を描画する
- **前提**
  - レスポンスオブジェクトで `status` / `render` をモック化する。
- **操作**
  - 登録済みハンドラーを実行する。
- **結果**
  - `screen/login` がフォーム設定を含む表示データで描画される。
